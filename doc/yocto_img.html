Yocto/Img

Utilities for loading/saving images, mostly to avoid linking problems and code duplication across other yocto libraries. Functions as a wrapper to other libraries.

This library depends in yocto_math.h, stb_image.h, stb_image_write.h stb_image_resize.h.

Usage

  1. load images with load_image()
  2. save images with save_image()
  3. resize images with resize_image()

History

  • v 0.1: added whole image functions
  • v 0.1: initial release
