Yocto/Gui

A set of utilities for windows= management and immediate mode widgets using GLFW and ImGui. This is just a straightfoward wrapper with the main benefit of integrating ImGui in GLFW without explicit application support by modying GLFW callbacks. Also provides easy to use ImGui wrappers.

This library depends in yocto_math.h The library depends on GLEW for OpenGL functions on Windows and Linux, GLFW for window management and dear ImGui for widgets.

History

  • v 0.3: changed image widget sizing
  • v 0.2: added widgets explicit IDs
  • v 0.1: added more widgets
  • v 0.0: initial release split from yocto_glu
